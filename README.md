# Item to Check

A simple Python script that checks if an item exists in a packing list and finds its index.

## Files

- **script.py** - Practice script for working with lists, strings, and basic Python operations including suitcase packing examples
- **solution.py** - Main script that checks if an item exists in a packing list and finds its index

## Usage

### Run the practice script:
```bash
python script.py
```

### Run the solution script:
```bash
python solution.py
```

## Output (solution.py)

```
Is the item packed? True
Item index: 2
```
